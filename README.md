## A small project to understand and use the 'Zustand' management library for React.

## What is Zustand?

Zustand is a small, fast, and scalable state management library for React. It provides a simple and minimalistic way to manage global state without the complexity of tools like Redux or Context API. 

Zustand uses a central store, created with a hook, where you can easily store and update your application state. It supports powerful features like middleware, persistence, and devtools — while keeping the developer experience clean and easy to use.

The word *"Zustand"* is German and means *"state"*, reflecting the library's main purpose: managing state in React applications.
